# Ai Cai De Ren

## Developer Setup

#### install npm & dependencies

```bash
$ cd project/              
$ nvm use 6.9.1                 
$ sudo npm -g install gulp      
$ npm install
```

### Building tasks

`gulp test` unit test.

`gulp test:coverage` test coverage report.

`gulp dist` task will compile the project and pact it into dist/

### Running Locally

`gulp serve`

Then visit [http://localhost:8000](http://localhost:8000)
